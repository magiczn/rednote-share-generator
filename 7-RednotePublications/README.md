# 小红书文字分享生成器

一个可以将小红书文案转换成精美 3:4 比例图片的在线工具，适合发布到小红书平台。

## 功能特点

- 📱 **3:4 比例页面**：完美适配小红书推荐尺寸
- ✨ **智能分页**：根据内容长度自动分页，每页无滚动条
- 🎨 **多种主题**：4 种配色主题可选（白、暖白、灰、暗黑）
- 🔤 **字体可调**：支持 14-20px 字体大小调整
- 💾 **一键下载**：支持下载单页或全部页面为 PNG 图片
- 📅 **自动日期**：自动添加当前日期

## 使用方法

1. 直接在浏览器中打开 `index.html` 文件
2. 在左侧编辑区输入标题和文案内容
3. 选择合适的主题色和字体大小
4. 在右侧预览生成的效果
5. 点击下载按钮保存为 PNG 图片

## 技术栈

- HTML5 + CSS3
- Tailwind CSS（CDN）
- html2canvas（截图功能）

## 文件说明

- `index.html` - 主应用文件
- `example.txt` - 示例文案参考

## 注意事项

- 文案会自动分页，确保每页内容都能完整显示
- 截图质量为 2x 高清，适合发布
- 支持中文内容，自动处理换行和标点

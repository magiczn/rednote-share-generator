# ToDo - 待办事项应用

一个简约清新的待办事项Web应用，帮助你高效管理日常任务。

## 功能特性

- **添加任务** - 在输入框中输入任务内容，按回车或点击添加按钮
- **完成任务** - 点击复选框标记任务完成/未完成
- **标记重要** - 点击星标将任务标记为重要
- **编辑任务** - 双击任务内容可直接编辑
- **删除任务** - 悬停任务后点击删除按钮
- **任务筛选** - 支持全部/未完成/已完成/重要四种筛选视图
- **清除已完成** - 一键清除所有已完成的任务
- **数据持久化** - 自动保存到浏览器本地，刷新不丢失

## 使用方式

直接在浏览器中打开 `index.html` 即可使用，无需任何构建步骤。

```bash
# 使用本地服务器预览（推荐）
python3 -m http.server 8080
# 然后访问 http://localhost:8080
```
